# hiPSC-CM-patch-RNAseq-analysis

Bioinformatics Final Project: Bulk RNA-seq reanalysis of GSE318811 for ECM/fibrosis-related transcriptomic changes after hiPSC-CM patch transplantation.

This repository contains the R scripts, selected dataset files, and analysis results used in our final project report.

## Project overview

This project reanalyzes the GSE318811 bulk RNA-seq dataset to investigate patch-associated host transcriptomic changes related to ECM remodeling, fibrosis, immune-inflammatory response, and wound repair. The analysis focuses on infarct zone (IZ) samples across multiple time points and border zone (BZ) samples at 1 week.

## Repository contents

* `Dataset/`: selected input dataset files used for analysis.
* `R Scripts/`: R scripts for DESeq2 analysis, PCA, volcano plots, enrichment analysis, Metascape processing, and heatmap visualization.
* `Results/`: DEG tables, enrichment outputs, and representative figures.
